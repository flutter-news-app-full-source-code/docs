After evaluating the toolkit with a local setup, the next step is to go live. These guides are for developers who have acquired a commercial license and are ready to transition each component of the toolkit from their local machine to a live, production environment.

<Steps>
1.  **Deploy the API Server**

    Learn how to configure and deploy the Dart Frog backend to a production server.

    [Go to API Deployment Guide &rarr;](/docs/api-server/deployment/)

2.  **Deploy the Web Dashboard**

    Follow the steps to build and deploy the Flutter web dashboard to your preferred hosting provider.

    [Go to Dashboard Deployment Guide &rarr;](/docs/web-dashboard/deployment/)

3.  **Deploy the Mobile Client**

    Prepare and build your Flutter mobile app for release on the Apple App Store and Google Play Store.

    [Go to Mobile App Deployment Guide &rarr;](/docs/mobile-client/deployment/)
</Steps>
